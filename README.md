jeu de dévinette 
L'ordinateur génère un nombre aléatoire entre 1 et 100 que vous devez déviner

règles du jeu
- L'ordinateur choisit un nombre aléatoire entre 1 et 100
- Vous avez droit à 10 essais pour trouver ce nombre
- Après chaque essai un indice vous est donné :
- Nombre proposé plus grand si le nombre que vous avez choisi est plus grand que le nombre secret 
- Nombre proposé plus petit  si le nombre que vous avez choisi est plus petit que le nombre secret 
- Vous gagnez si vous trouvez le bon nombre avant 10 essais
- Vous perdez si vous utilisez tous vos essais sans trouver le nombre secret 

Prérequis
- Java JDK 11 
- IntelliJ IDEA
